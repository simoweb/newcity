New City

HTML Template

  • created: 08/20/2014
  • latest update: 08/18/2014
  • by: Simone Puliti
  • site: simowebdesigner.it
  • email: simone.puliti@gmail.com

New City is a multipurpose Responsive HTML template suitable for any creative business agency. It’s fully responsive, has moulded structure. You will define what you want to use on your website. Optimized and intuitive code using the latest HTML5 and CSS3 technologies allows to make changes in the page structure of the template.
